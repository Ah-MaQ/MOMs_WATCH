1. 프로젝트 디렉토리로 이동 
새로운 컴퓨터에서 터미널 또는 명령 프롬프트를 열고, 프로젝트 디렉토리로 이동
[cd <프로젝트 디렉토리>]

2. 필요한 모듈 설치
[npm install] 명령어 실행
= package.json 파일에 명시된 모든 종속성 설치.
-> npm install 성공적으로 마치면 nod_modules 폴더 생성

3. 개발 서버 시작 (옵션)
개발 서버를 사용하고자 할 경우 npm start 명령어 실행
[npm start]


4. Chrome 확장 프로그램 로드


** 만약 개발 서버를 사용하지 않고, 크롬 확장 프로그램을 직접 로드하여 사용하는 경우 npm start 명령어는 실행하지 않아도 되지만 npm install은 반드시 실행해야 함
